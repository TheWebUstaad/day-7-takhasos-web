🧠 Ramzan Iftar Time
Day 6 & 7 Task

📌 Task Title

Ek Iftar Page Design karein — Ghar jesa dastarkhwan feel aaye!

🕌 Task Description

Aapko ek HTML page banana hai jismein Ramzan ki shaam ka Iftar Time display ho.
Page dekh kar Islamic aur sukoon bhara feel aaye — design simple aur pyara ho 🎨

✅ Page Sections

  • Top Banner: 🌙 Ramzan Mubarak – Iftar Time Table
  • Iftar Table: 3 items (image + name + Sunnat ya dua)
  • Sunnat List: Khajoor, Pani, Doodh, Phal, Nariyal
  • Quote Section: "Jo shakhs kisi roza daar ko iftar karwaye..."
  • Footer Note: Designed with ❤️ for Ramzan

🎨 Styling Tips

  • Background: soft green ya golden
  • Font: Arial ya clean readable font
  • Headings colorful aur bold hon
  • Flexbox sirf iftar items ke liye use karein
  • Images size: 100x100px
  • Quote italic aur center aligned ho

🤲 Islamic Feel Ideas

Images aise hon jo Islamic lagain (e.g. khajoor, sharbat, dastarkhwan).
Dua likhna chaho to ye use karo: Allahumma inni laka sumtu...

📦 Submission

  • Sirf ek HTML file banani hai: ramzan-iftar.html
  • Internal CSS use karein — <style> tag head ke andar
  • Video bana kar share karein
